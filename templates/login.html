What is a Jupyter Notebook?

Jupyter Notebook is an open-source web application that allows you to create and share documents containing live code, equations, visualizations, and narrative text. Common uses include data cleaning and transformation, numerical simulation, statistical modeling, data visualization, and machine learning.

What is JupyterHub?

JupyterHub brings the power of notebooks to groups of users. It gives students and researchers access to computational environments and resources without requiring local installation or maintenance. All computation runs on shared, centrally managed infrastructure.

Who can use this service?

Open Jupyter is available to UBC students, instructors and staff.

How do I distribute course content to students?

We recommend using nbgitpuller to distribute content to students. You can construct an nbgitpuller link and share it with your class. When students click it, the course content will be pulled directly into their home directories.

What packages & libraries are available?

Several notebook environments are available depending on your course profile, including Python (SciPy), TensorFlow, Julia, and R/RStudio. For a full list of installed packages in each environment, see the UBC Jupyter Images repository.

Can I install my own packages?

Yes! In a Jupyter Notebook, use !pip install package-name for Python or install.packages() for R. Note that user-installed packages do not persist across server restarts — add the install command to the top of your notebook so it runs each session.

Can a package be installed for all my students?

Most likely! Contact LTIC support with your request and we can add it to the course image for the next deployment.
